# React Learning Environment

A challenge-based arena for learning React from scratch.

## Getting Started

```bash
npm install
npm run dev
```

Open the URL shown in your terminal (usually `http://localhost:5173`).

## How It Works

| Area | Purpose |
|------|---------|
| Left sidebar | Navigate between challenges |
| Description panel | Task, hints, and acceptance criteria |
| Preview pane | Your code renders here live (hot reload) |
| "Peek at Solution" | Only after genuinely trying! |

## Workflow

1. Select a challenge in the sidebar
2. Open the file shown in the description panel (e.g. `src/challenges/01-hello-jsx/Challenge.jsx`)
3. Read the task and TODOs inside the file
4. Edit and save — the preview updates instantly
5. Check the acceptance criteria — if they all pass, move on

## Challenges

| # | Topic | Concepts |
|---|-------|---------|
| 01 | Hello JSX | JSX syntax, returning markup |
| 02 | Your First Component | Component functions, composition |
| 03 | Props | Passing data, reusability |
| 04 | useState | State, hooks, re-rendering |
| 05 | Event Handling | onClick, event handlers |
| 06 | Conditional Rendering | Ternary, &&, conditional JSX |
| 07 | Lists & Keys | .map(), key prop, dynamic data |

## Tips

- **Break things on purpose** — the only way to learn is to experiment
- **Read error messages** — React's errors are descriptive and helpful
- **Check the console** — open DevTools with `Cmd+Option+I`
- **Use hints** — they guide without giving it away
